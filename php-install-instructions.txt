Инструкция по установке PHP на Windows и добавлению в PATH

1. Скачайте PHP:
   - Перейдите на официальный сайт: https://windows.php.net/download/
   - Выберите последнюю версию PHP (например, PHP 8.x) в разделе "Thread Safe" (x64 или x86 в зависимости от вашей системы).
   - Скачайте ZIP-архив.

2. Распакуйте архив:
   - Распакуйте содержимое архива в удобную папку, например: C:\php

3. Добавьте PHP в системный PATH:
   - Откройте "Панель управления" → "Система" → "Дополнительные параметры системы" → "Переменные среды".
   - В разделе "Системные переменные" найдите переменную "Path" и нажмите "Изменить".
   - Добавьте новый путь: C:\php (путь к папке с php.exe).
   - Нажмите OK, чтобы сохранить изменения.

4. Проверьте установку:
   - Откройте новую командную строку (cmd).
   - Введите команду:
     php -v
   - Вы должны увидеть информацию о версии PHP.

5. Запуск встроенного PHP-сервера:
   - В командной строке перейдите в папку вашего проекта, например:
     cd d:\АН ФАТТОРИЯ
   - Запустите сервер командой:
     php -S localhost:8000
   - Откройте в браузере:
     http://localhost:8000/наши%20обьекты.html

Если возникнут вопросы или проблемы, обращайтесь за помощью.
